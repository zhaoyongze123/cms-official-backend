# Roadmap

本 Roadmap 用于描述项目面向开源展示和后续迭代的方向，不替代具体任务拆分文档。

## 已完成

- Django CMS 后台与内容模型基础能力
- FastAPI AI / RAG 服务骨架
- Next.js Studio 工作台
- Next.js Public Web 对外站点
- Docker Compose 开发与生产环境
- GitHub Actions `main` 自动部署工作流
- 生产 Nginx 反向代理模板
- SEO 监控入口与真实 API 取数链路

## 近期计划

- 补充公开站页面截图与功能演示
- 完善 Django API 文档与示例请求
- 增加部署故障排查手册
- 扩展 SEO 监控数据源接入说明
- 梳理 Studio 编辑器的用户操作指南

## 中期计划

- 强化 AI 审核与 RAG 检索链路
- 完善发布前 SEO 检查与阻断策略
- 增加文章版本、审计记录与回滚能力
- 提升 Public Web 的缓存与性能策略
- 增加端到端验收测试

## 长期方向

- 多站点管理
- 多语言内容发布
- 插件化内容模块
- 更完整的权限模型
- 面向生产环境的可观测性仪表盘
